# ClowtheX - تطبيق إدارة محل الملابس

<div dir="rtl">

تطبيق Flutter احترافي لإدارة محلات الملابس الرجالية. يعمل بالكامل بدون إنترنت.

## الميزات

### 🛒 نقطة البيع (POS)
- إضافة منتجات بالاسم أو مسح الباركود
- حساب الإجمالي تلقائياً مع الخصم والضريبة
- دعم الدفع نقداً وبطاقة وتحويل
- طباعة الفاتورة مباشرة

### 📦 إدارة المخزون
- إضافة/تعديل/حذف المنتجات
- تصنيف المنتجات بالفئات
- تنبيهات المخزون المنخفض والنافد
- مسح الباركود لإضافة المنتجات
- بحث وفلترة متقدمة

### 📊 التقارير
- تقارير المبيعات اليومية/الشهرية
- رسوم بيانية للمبيعات
- أكثر المنتجات مبيعاً
- تقرير المخزون الكامل

### 📤 الاستيراد والتصدير
| العملية | الصيغة | الوصف |
|---------|--------|-------|
| تصدير نسخة احتياطية | JSON | كامل البيانات |
| استيراد نسخة احتياطية | JSON | استعادة كاملة |
| تصدير المنتجات | Excel | قائمة المنتجات كاملة |
| تصدير المبيعات | Excel | سجل الفواتير |
| استيراد المنتجات | Excel | رفع كميات كبيرة |
| طباعة الفاتورة | PDF | مباشرة للطابعة |
| طباعة تقرير المخزون | PDF | مباشرة للطابعة |
| طباعة تقرير المبيعات | PDF | مباشرة للطابعة |

### ⚙️ الإعدادات
- اسم المحل والعنوان والهاتف
- رمز العملة ونسبة الضريبة
- حد تنبيه المخزون

## التقنيات المستخدمة

| الغرض | الحزمة |
|-------|--------|
| قاعدة البيانات | sqflite (SQLite) |
| إدارة الحالة | Provider |
| الباركود/QR | mobile_scanner, qr_flutter |
| PDF وطباعة | pdf, printing |
| Excel | excel |
| مشاركة الملفات | share_plus, file_picker |
| الرسوم البيانية | fl_chart |
| الواجهة | Material 3 + RTL |

## كيفية البناء

### متطلبات
- Flutter SDK >= 3.22.0
- Android SDK
- Java 17

### خطوات
```bash
# 1. تثبيت الحزم
flutter pub get

# 2. بناء APK
flutter build apk --release

# 3. أو بناء مع تقسيم حسب المعمارية
flutter build apk --release --split-per-abi
```

ملف APK سيكون في: `build/app/outputs/flutter-apk/`

### بناء تلقائي عبر GitHub Actions
ادفع الكود إلى GitHub، سيتم بناء APK تلقائياً في:
`Actions → Build APK → Artifacts`
